# 🔋 Lite YT & Data Saver

A powerful, ultra-lightweight desktop browser for YouTube, specifically designed to save mobile data and system resources.

## 🚀 Why Use Lite YT?

- **📉 Data Saving**: Automatically forces video quality to **360p** (Medium) or lower to significantly reduce data usage on mobile hotspots.
- **⚡ Performance First**: Built with **Tauri and Rust**, using up to 10x less RAM and CPU than Google Chrome or Edge.
- **🎮 Streamer Optimized**: Ideal for low-end PCs or streamers who need every bit of FPS for their games.
- **🤫 Quiet UI**: No cluttered layouts—just a clean, focused YouTube experience.

## 🛠️ Built With

- **Tauri v2**: Minimal resource footprint.
- **Rust**: High-performance backend.
- **Vanilla JS**: Optimized data-saving injections.

## 🔧 Developer Setup

1. Clone the repository.
2. Run `npm install`.
3. Start in dev mode with `npm run tauri dev`.
